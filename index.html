SOME OF MY RECENT PROJECTS

DESIGN OF SWB MIMO ANTENNA FOR IOT APPLICATIONS

• Designed & Fabricated a Super Wide Band (SWB) Multi Input Multi Output (MIMO) Antenna with a very compact dimension of 45mm x 45mm x 1.6mm with a bandwidth range of 1GHz-15GHz.
• Designed a unique slot-loaded diagonal clipped square-shaped radiation patch and partial ground plane. To maximize isolation a train square plus shaped decoupling structure is embedded at both side of the substrate. Isolation is better than -25dB for the most of the frequency band.
• Circular Polarization, Polarization Diversity and Pattern Diversity is achieved at desired frequencies. The Envelope Correlation Coefficient (ECC) of the antenna is very low and Diversity Gain is high.
• Tools used - Ansys HFSS (High Frequency Structure Simulator), Origin Pro

SMART HEALTH MONITORING SYSTEM

• Designed and built a system to check and keep track of the user’s health and connect the users and connect with the admin (medical facilities) in an efficient method, to provide service with a better efficiency.
• The Individual devices for the users are developed using ESP-32S Development Board (ESP-WROOM-32) & MAX30100 I2C Pulse Oximeter Sensor
• Arduino is used to code the functionalities for the ESP32 and utilizing ESP32 Library, MAX30102 Library, & NTP-Client Library, the user’s data (BPM and SPO2 levels) are tracked and uploaded to the Google Firebase database.
• The web application is developed as a single page application using HTML, CSS, JavaScript, and ReactJS.
